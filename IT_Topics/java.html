What is Java?

Java is a high-level, class-based, object-oriented programming language designed to have as few implementation dependencies as possible. It is one of the most popular programming languages used for building web applications, mobile applications, and large-scale enterprise systems.

Features of Java

  • Object-Oriented: Java follows the object-oriented programming paradigm, which promotes code reuse and modularity.
  • Platform-Independent: Java applications are designed to be portable across different operating systems. The Java Virtual Machine (JVM) allows Java programs to run on any device or operating system that has the JVM installed.
  • Robust: Java provides strong memory management and exception handling features, which help in building reliable and error-free programs.
  • Multithreaded: Java supports multithreading, which allows multiple threads to run concurrently, enhancing the performance of applications.
  • Secure: Java includes several built-in security features such as the sandboxing of applets and a robust security manager.

Basic Syntax of Java

Java programs are written in classes. Here’s a simple example of a Java class:

public class HelloWorld {
                    public static void main(String[] args) {
                        System.out.println("Hello, World!");
                    }
                }

This example demonstrates a basic Java program that prints "Hello, World!" to the console:

public class HelloWorld {

                        System.out.println("Hello, World!");
                    }
                }

Output: Hello, World!

Java Introduction

History of Java

Java was developed by Sun Microsystems, led by James Gosling, in the mid-1990s. The first version, Java 1.0, was released in 1996. Java was designed with the principle of "write once, run anywhere" in mind, allowing developers to create applications that can run on any device with a Java Virtual Machine (JVM). In 2010, Oracle Corporation acquired Sun Microsystems and has since maintained and developed the Java platform.

Introduction to Java

Java is a high-level, object-oriented programming language known for its portability across platforms, performance, and security features. It is used in a wide range of applications, from web development to mobile applications (especially Android) and large-scale enterprise systems. Java's syntax is influenced by C and C++, making it familiar to many developers.

How Java Works

Java programs are compiled into bytecode, which is an intermediate form of code that runs on the Java Virtual Machine (JVM). The JVM interprets or compiles this bytecode into native machine code that is executed by the host system. This process allows Java programs to be platform-independent, as the same bytecode can run on any system with a compatible JVM.

Java Syntax

Java Statements

Java statements are the building blocks of Java programs. They define the actions to be performed and typically end with a semicolon (;). Statements include expressions, variable declarations, method calls, and control flow statements.

                    int number = 10;
                    System.out.println("Number: " + number);

In this example, an integer variable number is declared and initialized to 10. The System.out.println statement is used to print the value of the variable to the console.

Java Comments

Comments in Java are used to explain the code and are ignored during execution. There are three types of comments in Java:

  • Single-Line Comments: Start with // and continue to the end of the line.
  • Multi-Line Comments: Enclosed between /* and */ and can span multiple lines.
  • Javadoc Comments: Used for generating documentation. Start with /** and end with */.
                    // This is a single-line comment
                    /* This is a multi-line comment
                       that spans multiple lines */
                    /**
                     * This is a Javadoc comment
                     * used for generating documentation
                     */
                    System.out.println("Hello, World!");

The // starts a single-line comment, /* ... */ is used for multi-line comments, and /** ... */ is used for Javadoc comments.

Java Output and Printing

Printing to the Console

In Java, you can print text and data to the console using the System.out.println() and System.out.print() methods. The println() method prints a line of text and moves to the next line, while the print() method prints text without moving to the next line.

                    System.out.println("Hello, World!"); // Prints "Hello, World!" and moves to a new line
                    System.out.print("Hello, "); // Prints "Hello, " without moving to a new line
                    System.out.print("World!"); // Prints "World!" on the same line
                    System.out.println(); // Moves to the next line

In this example, System.out.println() prints "Hello, World!" and then moves to the next line, while System.out.print() prints "Hello, " and "World!" on the same line.

Formatting Output

You can format output using System.out.printf(), which allows you to format strings with placeholders. Placeholders are replaced with the provided values.

                    int age = 25;
                    String name = "Alice";
                    System.out.printf("Name: %s, Age: %d%n", name, age);

In this example, System.out.printf() prints "Name: Alice, Age: 25", where %s is a placeholder for a string and %d is a placeholder for an integer. The %n represents a newline character.

Printing Numbers

Java provides various methods to print different types of numbers. You can print integers, floating-point numbers, and formatted numbers using System.out.println() and System.out.printf().

                    int number = 10;
                    double price = 99.99;
                    System.out.println("Integer: " + number); // Prints an integer
                    System.out.printf("Price: %.2f%n", price); // Prints a floating-point number with two decimal places

In this example, System.out.println() prints an integer, and System.out.printf() prints a floating-point number with two decimal places.

Java Comments

Introduction to Comments

Comments in Java are used to annotate code, making it easier to understand. They are ignored by the Java compiler and have no effect on the execution of the program. Comments are useful for explaining code functionality, adding notes, and making code more maintainable.

Single-Line Comments

Single-line comments begin with //. Everything following // on that line is considered a comment and is ignored by the compiler.

                    // This is a single-line comment
                    int x = 10; // This comment is at the end of a line

Multi-Line Comments

Multi-line comments begin with /* and end with */. They can span multiple lines and are used for longer explanations or commenting out blocks of code.

                    /*
                    This is a multi-line comment
                    It can span multiple lines
                    */
                    int y = 20;

Documentation Comments

Documentation comments, also known as Javadoc comments, begin with /** and end with */. They are used to generate HTML documentation for Java classes and methods. They typically include descriptions of the class or method, parameters, and return values.

                    /**
                    * This is a documentation comment
                    * It describes the purpose of the method
                    * @param name The name to be greeted
                    * @return A greeting message
                    */
                    public String greet(String name) {
                        return "Hello, " + name;
                    }

Java Variables

Introduction to Variables

In Java, variables are used to store data that can be used and manipulated throughout a program. Each variable has a specific type, which determines the kind of data it can hold. Variables are essential for performing operations, storing values, and controlling the flow of a program.

Declaration and Initialization

To use a variable in Java, you must first declare it by specifying its type and name. Optionally, you can also initialize it with a value.

                    // Declaration of a variable
                    int age;
                    
                    // Initialization of a variable
                    age = 30;
                    
                    // Declaration and initialization in one statement
                    int salary = 50000;

Types of Variables

Java variables can be categorized into different types based on their scope and lifespan:

  • Local Variables: Declared within a method or block and accessible only within that scope.
  • Instance Variables: Declared within a class but outside any method. Each instance of the class has its own copy.
  • Static Variables: Declared with the static keyword. Shared among all instances of the class.

Example of Variable Usage

Here is a simple example of different types of variables:

                    public class Main {
                        // Static variable
                        static int staticVar = 100;
                        
                        // Instance variable
                        int instanceVar;
                        
                        public void method() {
                            // Local variable
                            int localVar = 10;
                            
                            System.out.println("Local Variable: " + localVar);
                            System.out.println("Instance Variable: " + instanceVar);
                            System.out.println("Static Variable: " + staticVar);
                        }
                    }

Java Data Types

Introduction to Data Types

In Java, data types define the kind of data that can be stored and manipulated within a program. Java is a statically-typed language, meaning that variables must be declared with a specific data type, and this type cannot change during runtime.

Primitive Data Types

Java has eight built-in primitive data types:

  • byte: 8-bit integer, range from -128 to 127.
  • short: 16-bit integer, range from -32,768 to 32,767.
  • int: 32-bit integer, range from -2^31 to 2^31 - 1.
  • long: 64-bit integer, range from -2^63 to 2^63 - 1.
  • float: 32-bit floating-point number.
  • double: 64-bit floating-point number.
  • char: 16-bit Unicode character, range from '\u0000' to '\uffff'.
  • boolean: Represents true or false values.

Reference Data Types

Reference data types are used to refer to objects and arrays. Unlike primitive data types, they do not hold the actual data but instead hold a reference to the data. Common reference data types include:

  • String: Represents a sequence of characters.
  • Arrays: Represents a collection of elements of the same type.
  • Classes: Custom data types defined by the user.
  • Interfaces: Abstract types that can be implemented by classes.

Example of Data Types

Here’s an example demonstrating the use of various data types in Java:

                    public class DataTypesExample {
                        public static void main(String[] args) {
                            // Primitive data types
                            byte b = 100;
                            short s = 10000;
                            int i = 100000;
                            long l = 10000000000L;
                            float f = 10.5f;
                            double d = 20.5;
                            char c = 'A';
                            boolean bool = true;
                            
                            // Reference data type
                            String str = "Hello, Java!";
                            
                            System.out.println("Byte: " + b);
                            System.out.println("Short: " + s);
                            System.out.println("Int: " + i);
                            System.out.println("Long: " + l);
                            System.out.println("Float: " + f);
                            System.out.println("Double: " + d);
                            System.out.println("Char: " + c);
                            System.out.println("Boolean: " + bool);
                            System.out.println("String: " + str);
                        }
                    }

Java Type Casting

Introduction to Type Casting

Type casting in Java is the process of converting one data type into another. It is necessary when you need to convert variables from one type to another, especially when dealing with primitive data types or objects of different types. Java supports two types of casting: implicit casting (automatic) and explicit casting (manual).

Implicit Casting (Automatic Type Conversion)

Implicit casting, also known as automatic type conversion, occurs when you assign a smaller data type to a larger data type. This is done automatically by the Java compiler, as there is no risk of data loss.

                    
                    // Example of implicit casting
                    int myInt = 9;
                    double myDouble = myInt;  // int is automatically converted to double
                    System.out.println("Double value: " + myDouble);  // Output will be 9.0

Explicit Casting (Manual Type Conversion)

Explicit casting, also known as manual type conversion, is required when you want to convert a larger data type to a smaller data type. This must be done manually and can result in data loss if the value cannot be accurately represented in the smaller data type.

                    
                    // Example of explicit casting
                    double myDouble = 9.78;
                    int myInt = (int) myDouble;  // double is manually converted to int
                    System.out.println("Integer value: " + myInt);  // Output will be 9

Type Casting Objects

In addition to primitive types, you can also cast objects between related classes. For example, if you have a parent class reference pointing to a child class object, you can cast it to the child class type. However, casting between unrelated classes will result in a ClassCastException.

                    
                    class Animal {
                        public void makeSound() {
                            System.out.println("Some sound");
                        }
                    }
                    
                    class Dog extends Animal {
                        public void makeSound() {
                            System.out.println("Bark");
                        }
                    }
                    
                    public class Main {
                        public static void main(String[] args) {
                            Animal myAnimal = new Dog(); // Implicit casting
                            myAnimal.makeSound();  // Output will be "Bark"
                            
                            Dog myDog = (Dog) myAnimal; // Explicit casting
                            myDog.makeSound();  // Output will be "Bark"
                        }
                    }

Java Operators

Introduction to Operators

Operators in Java are special symbols used to perform operations on variables and values. They can be categorized into several types, including arithmetic, relational, logical, bitwise, and more. Operators are fundamental in Java programming, allowing you to manipulate data and control the flow of execution.

Arithmetic Operators

Arithmetic operators are used to perform basic mathematical operations. They include:

  • + (Addition): Adds two operands.
  • - (Subtraction): Subtracts the second operand from the first.
  • * (Multiplication): Multiplies two operands.
  • / (Division): Divides the first operand by the second.
  • % (Modulus): Returns the remainder of a division operation.
                    
                    int a = 10;
                    int b = 5;
                    System.out.println("a + b = " + (a + b));  // Output: 15
                    System.out.println("a - b = " + (a - b));  // Output: 5
                    System.out.println("a * b = " + (a * b));  // Output: 50
                    System.out.println("a / b = " + (a / b));  // Output: 2
                    System.out.println("a % b = " + (a % b));  // Output: 0

Relational Operators

Relational operators are used to compare two values. They return a boolean result:

  • == (Equal to): Checks if two values are equal.
  • != (Not equal to): Checks if two values are not equal.
  • > (Greater than): Checks if the first value is greater than the second.
  • < (Less than): Checks if the first value is less than the second.
  • >= (Greater than or equal to): Checks if the first value is greater than or equal to the second.
  • <= (Less than or equal to): Checks if the first value is less than or equal to the second.
                    
                    int a = 10;
                    int b = 5;
                    System.out.println("a == b: " + (a == b));  // Output: false
                    System.out.println("a != b: " + (a != b));  // Output: true
                    System.out.println("a > b: " + (a > b));  // Output: true
                    System.out.println("a < b: " + (a < b));  // Output: false
                    System.out.println("a >= b: " + (a >= b));  // Output: true
                    System.out.println("a <= b: " + (a <= b));  // Output: false

Logical Operators

Logical operators are used to combine multiple boolean expressions and return a boolean result:

  • && (Logical AND): Returns true if both operands are true.
  • || (Logical OR): Returns true if at least one operand is true.
  • ! (Logical NOT): Reverses the boolean value.
                    
                    boolean a = true;
                    boolean b = false;
                    System.out.println("a && b: " + (a && b));  // Output: false
                    System.out.println("a || b: " + (a || b));  // Output: true
                    System.out.println("!a: " + (!a));  // Output: false

Bitwise Operators

Bitwise operators perform operations on the binary representations of integers:

  • & (Bitwise AND): Performs a bitwise AND operation.
  • | (Bitwise OR): Performs a bitwise OR operation.
  • ^ (Bitwise XOR): Performs a bitwise XOR operation.
  • ~ (Bitwise NOT): Performs a bitwise NOT operation.
  • << (Left shift): Shifts bits to the left.
  • >> (Right shift): Shifts bits to the right.
                    
                    int a = 5;   // 0101 in binary
                    int b = 3;   // 0011 in binary
                    System.out.println("a & b: " + (a & b));  // Output: 1 (0001 in binary)
                    System.out.println("a | b: " + (a | b));  // Output: 7 (0111 in binary)
                    System.out.println("a ^ b: " + (a ^ b));  // Output: 6 (0110 in binary)
                    System.out.println("~a: " + (~a));  // Output: -6 (inverted bits)
                    System.out.println("a << 1: " + (a << 1));  // Output: 10 (1010 in binary)
                    System.out.println("a >> 1: " + (a >> 1));  // Output: 2 (0010 in binary)

Assignment Operators

Assignment operators are used to assign values to variables and perform operations simultaneously:

  • = (Assignment): Assigns a value to a variable.
  • += (Add and assign): Adds a value to a variable and assigns the result.
  • -= (Subtract and assign): Subtracts a value from a variable and assigns the result.
  • *= (Multiply and assign): Multiplies a variable by a value and assigns the result.
  • /= (Divide and assign): Divides a variable by a value and assigns the result.
  • %= (Modulus and assign): Takes the modulus of a variable by a value and assigns the result.
                    
                    int a = 10;
                    a += 5;  // Equivalent to a = a + 5
                    System.out.println("a += 5: " + a);  // Output: 15
                    a -= 3;  // Equivalent to a = a - 3
                    System.out.println("a -= 3: " + a);  // Output: 12
                    a *= 2;  // Equivalent to a = a * 2
                    System.out.println("a *= 2: " + a);  // Output: 24
                    a /= 4;  // Equivalent to a = a / 4
                    System.out.println("a /= 4: " + a);  // Output: 6
                    a %= 4;  // Equivalent to a = a % 4
                    System.out.println("a %= 4: " + a);  // Output: 2

Java Strings

Introduction to Strings

In Java, a String is a sequence of characters used to represent text. The String class in Java is part of the java.lang package and provides various methods to manipulate and work with text. Strings are immutable, meaning once a string is created, it cannot be changed. Any modifications to a string result in the creation of a new string.

Creating Strings

Strings can be created using string literals or the String class constructor:

                    
                    // Using string literals
                    String str1 = "Hello, World!";
                    
                    // Using the String constructor
                    String str2 = new String("Hello, Java!");

String Concatenation

Strings can be concatenated using the + operator or the concat() method:

                    
                    String firstName = "John";
                    String lastName = "Doe";
                    
                    // Using the + operator
                    String fullName = firstName + " " + lastName;
                    
                    // Using the concat() method
                    String fullNameConcat = firstName.concat(" ").concat(lastName);
                    
                    System.out.println("Full Name: " + fullName);
                    System.out.println("Full Name (concat): " + fullNameConcat);

String Methods

Java provides a variety of methods to manipulate strings. Here are a few commonly used methods:

  • length(): Returns the length of the string.
  • charAt(int index): Returns the character at the specified index.
  • substring(int beginIndex, int endIndex): Returns a substring from the specified indices.
  • toLowerCase(): Converts all characters in the string to lowercase.
  • toUpperCase(): Converts all characters in the string to uppercase.
  • trim(): Removes leading and trailing whitespace.
  • replace(char oldChar, char newChar): Replaces occurrences of a specified character with a new character.
  • split(String regex): Splits the string into an array based on a delimiter.
                    
                    String str = "  Java Programming  ";
                    
                    // Getting the length of the string
                    int length = str.length();
                    
                    // Character at index 2
                    char ch = str.charAt(2);
                    
                    // Substring from index 2 to 6
                    String subStr = str.substring(2, 6);
                    
                    // Converting to lowercase
                    String lowerStr = str.toLowerCase();
                    
                    // Converting to uppercase
                    String upperStr = str.toUpperCase();
                    
                    // Trimming whitespace
                    String trimmedStr = str.trim();
                    
                    // Replacing characters
                    String replacedStr = str.replace('a', 'A');
                    
                    // Splitting the string
                    String[] words = str.split(" ");
                    
                    System.out.println("Length: " + length);
                    System.out.println("Character at index 2: " + ch);
                    System.out.println("Substring: " + subStr);
                    System.out.println("Lowercase: " + lowerStr);
                    System.out.println("Uppercase: " + upperStr);
                    System.out.println("Trimmed: '" + trimmedStr + "'");
                    System.out.println("Replaced: " + replacedStr);
                    System.out.println("Words: " + java.util.Arrays.toString(words));

String Immutability

In Java, strings are immutable. This means that once a string is created, it cannot be changed. Instead, any modification to a string results in the creation of a new string. For example:

                    
                    String original = "Hello";
                    String modified = original.concat(" World");
                    
                    System.out.println("Original: " + original);  // Output: Hello
                    System.out.println("Modified: " + modified);  // Output: Hello World

Java Math

Introduction to Java Math

Java provides a built-in class Math in the java.lang package for performing mathematical operations. The Math class contains various methods for basic arithmetic, trigonometry, logarithms, and more.

Basic Math Operations

Here are some common mathematical operations provided by the Math class:

                    
                    // Addition

                    // Subtraction

                    // Multiplication

                    // Modulus

Mathematical Constants

The Math class provides several mathematical constants:

                    
                    double pi = Math.PI; // 3.141592653589793

                    // Sine of 45 degrees

                    // Cosine of 45 degrees

Java Booleans

Introduction to Booleans in Java

In Java, the boolean data type is used to represent true or false values. It is one of the simplest data types and is crucial for controlling the flow of a program through conditional statements and loops.

Boolean Variables

A boolean variable can hold one of two values: true or false. Here's how you declare and initialize boolean variables:

                    
                    boolean isJavaFun = true;
                    boolean isCold = false;

Boolean Expressions

Boolean expressions evaluate to either true or false. They are often used in conditional statements and loops:

                    
                    int age = 20;
                    boolean isAdult = (age >= 18); // true
                    
                    boolean isNotAdult = (age < 18); // false

Using Booleans in Conditional Statements

Boolean values are commonly used in if-else statements and other control flow constructs:

                    
                    int number = 10;
                    
                    if (number > 0) {
                        System.out.println("The number is positive.");
                    } else {
                        System.out.println("The number is zero or negative.");
                    }

Boolean Operators

Java provides several boolean operators to perform logical operations:

  • && (logical AND): Returns true if both operands are true.
  • || (logical OR): Returns true if at least one operand is true.
  • ! (logical NOT): Returns the opposite boolean value of the operand.
                    
                    boolean a = true;
                    boolean b = false;
                    
                    boolean andResult = a && b; // false
                    boolean orResult = a || b;  // true
                    boolean notResult = !a;     // false

Boolean Methods

Java provides methods to work with boolean values, such as:

  • Boolean.parseBoolean(String s): Converts a string to a boolean value.
  • Boolean.valueOf(String s): Returns a Boolean object holding the value of the specified string.
  • Boolean.toString(boolean b): Returns a string representation of the specified boolean value.
                    
                    boolean parsedValue = Boolean.parseBoolean("true"); // true
                    Boolean booleanObject = Boolean.valueOf("false"); // false
                    String booleanString = Boolean.toString(true);    // "true"

Java If ... Else Statements

Introduction to If ... Else

The if and else statements in Java are used to execute different blocks of code based on certain conditions. They help in making decisions within a program and control the flow of execution.

Basic If Statement

The if statement evaluates a condition and executes a block of code if the condition is true:

                    
                    int number = 10;
                    
                    if (number > 0) {
                        System.out.println("The number is positive.");
                    }

Output: This code prints "The number is positive." if the value of number is greater than 0.

If ... Else Statement

The if ... else statement allows you to execute one block of code if a condition is true and another block of code if the condition is false:

                    
                    int number = -5;
                    
                    if (number > 0) {
                        System.out.println("The number is positive.");
                    } else {
                        System.out.println("The number is zero or negative.");
                    }

Output: This code prints "The number is zero or negative." because the value of number is not greater than 0.

If ... Else If ... Else Statement

The if ... else if ... else statement allows for multiple conditions to be checked in sequence:

                    
                    int number = 0;
                    
                    if (number > 0) {
                        System.out.println("The number is positive.");
                    } else if (number < 0) {
                        System.out.println("The number is negative.");
                    } else {
                        System.out.println("The number is zero.");
                    }

Output: This code prints "The number is zero." as it matches the else condition.

Nested If Statements

You can place an if statement inside another if statement to create nested conditions:

                    
                    int number = 10;
                    
                    if (number >= 0) {
                        if (number == 0) {
                            System.out.println("The number is zero.");
                        } else {
                            System.out.println("The number is positive.");
                        }
                    } else {
                        System.out.println("The number is negative.");
                    }

Output: This code prints "The number is positive." because the outer if condition is true and the inner if condition is false.

Using Logical Operators in If Statements

You can use logical operators like && (AND), || (OR), and ! (NOT) in conditions:

                    
                    int age = 25;
                    boolean hasLicense = true;
                    
                    if (age >= 18 && hasLicense) {
                        System.out.println("You are eligible to drive.");
                    } else {
                        System.out.println("You are not eligible to drive.");
                    }

Output: This code prints "You are eligible to drive." if both conditions are true.

Java While Loop

The while loop in Java is used to execute a block of code repeatedly as long as a specified condition remains true. It is a fundamental control flow statement that helps to repeat actions until a condition changes.

Basic While Loop

The basic syntax of a while loop is as follows:


                int count = 0;
                
                while (count < 5) {
                    System.out.println("Count is: " + count);
                    count++;
                }

Output: This code prints the value of count from 0 to 4. The loop continues as long as count is less than 5.

Infinite While Loop

A while loop can run indefinitely if the condition is always true. Be cautious with infinite loops to avoid crashing your program:


                while (true) {
                    System.out.println("This will run forever!");
                    // Ensure there is a break condition or a way to exit the loop
                }

Output: This code will continuously print "This will run forever!" until the loop is forcibly terminated or an exit condition is added.

While Loop with Break Statement

The break statement can be used to exit a while loop before the condition becomes false:


                int number = 0;
                
                while (number < 10) {
                    if (number == 5) {
                        break; // Exit the loop when number is 5
                    }
                    System.out.println("Number is: " + number);
                    number++;
                }

Output: This code prints the value of number from 0 to 4. When number reaches 5, the break statement exits the loop.

While Loop with Continue Statement

The continue statement can be used to skip the current iteration and proceed to the next iteration of the loop:


                int i = 0;
                
                while (i < 10) {
                    i++;
                    if (i % 2 == 0) {
                        continue; // Skip even numbers
                    }
                    System.out.println("Odd number: " + i);
                }

Output: This code prints only the odd numbers from 1 to 9, as the continue statement skips the even numbers.

Java Switch Statement

The switch statement in Java is used to execute one block of code among many, based on the value of an expression. It's often used as a more readable alternative to multiple if-else statements when dealing with discrete values.

Basic Switch Statement

The basic syntax of a switch statement involves specifying the expression to be evaluated, followed by one or more case labels and a final default label:


                int day = 3;
                
                switch (day) {
                    case 1:
                        System.out.println("Monday");
                        break;
                    case 2:
                        System.out.println("Tuesday");
                        break;
                    case 3:
                        System.out.println("Wednesday");
                        break;
                    case 4:
                        System.out.println("Thursday");
                        break;
                    case 5:
                        System.out.println("Friday");
                        break;
                    case 6:
                        System.out.println("Saturday");
                        break;
                    case 7:
                        System.out.println("Sunday");
                        break;
                    default:
                        System.out.println("Invalid day");
                        break;
                }

Output: This code prints "Wednesday" because the value of day is 3, which matches the case 3 label.

Switch with Multiple Cases

You can group multiple case labels together to execute the same block of code for different values:


                int month = 6;
                
                switch (month) {
                    case 1:
                    case 2:
                    case 12:
                        System.out.println("Winter");
                        break;
                    case 3:
                    case 4:
                    case 5:
                        System.out.println("Spring");
                        break;
                    case 6:
                    case 7:
                    case 8:
                        System.out.println("Summer");
                        break;
                    case 9:
                    case 10:
                    case 11:
                        System.out.println("Fall");
                        break;
                    default:
                        System.out.println("Invalid month");
                        break;
                }

Output: This code prints "Summer" because the value of month is 6, which matches the case 6 label.

Switch with Strings (Java 7 and Later)

Starting from Java 7, the switch statement can also work with String values:


                String dayOfWeek = "Friday";
                
                switch (dayOfWeek) {
                    case "Monday":
                        System.out.println("Start of the work week");
                        break;
                    case "Friday":
                        System.out.println("End of the work week");
                        break;
                    case "Saturday":
                    case "Sunday":
                        System.out.println("Weekend");
                        break;
                    default:
                        System.out.println("Invalid day");
                        break;
                }

Output: This code prints "End of the work week" because the value of dayOfWeek is "Friday", which matches the case "Friday" label.

Switch with Fall-Through Behavior

If a case does not have a break statement, execution will fall through to the next case:


                int number = 2;
                
                switch (number) {
                    case 1:
                        System.out.println("One");
                    case 2:
                        System.out.println("Two");
                    case 3:
                        System.out.println("Three");
                        break;
                    default:
                        System.out.println("Not 1, 2, or 3");
                        break;
                }

Output: This code prints "Two" and "Three" because there is no break statement after case 1 and case 2, leading to fall-through behavior.

Java For Loop

The for loop in Java is used to iterate over a block of code a specific number of times. It is particularly useful when the number of iterations is known beforehand.

Basic For Loop

The basic syntax of a for loop is as follows:


                for (initialization; condition; update) {
                    // Code to be executed
                }

Here, initialization is executed once at the start, condition is evaluated before each iteration, and update is executed after each iteration.

Example of Basic For Loop


                for (int i = 0; i < 5; i++) {
                    System.out.println("Iteration: " + i);
                }

Output: This code prints the value of i from 0 to 4. The loop runs 5 times, incrementing i by 1 each time.

For Loop with Multiple Variables

You can initialize and update multiple variables within a for loop:


                for (int i = 0, j = 10; i < 5; i++, j--) {
                    System.out.println("i: " + i + ", j: " + j);
                }

Output: This code prints the values of i from 0 to 4 and j from 10 to 6, with i incrementing and j decrementing each iteration.

Enhanced For Loop

The enhanced for loop (or "for-each" loop) is used to iterate over elements in an array or a collection:


                int[] numbers = {1, 2, 3, 4, 5};
                
                for (int number : numbers) {
                    System.out.println("Number: " + number);
                }

Output: This code prints each element of the numbers array.

For Loop with Break and Continue

The break statement can exit the loop early, and the continue statement can skip the current iteration and move to the next iteration:


                for (int i = 0; i < 10; i++) {
                    if (i == 5) {
                        break; // Exit the loop when i is 5
                    }
                    if (i % 2 == 0) {
                        continue; // Skip even numbers
                    }
                    System.out.println("Odd number: " + i);
                }

Output: This code prints odd numbers less than 5. When i reaches 5, the break statement exits the loop.

Java Break and Continue Statements

The break and continue statements in Java control the flow of loops. They provide ways to manage how and when loops should terminate or skip iterations.

Break Statement

The break statement is used to exit from a loop prematurely. When break is encountered, the loop is terminated immediately, and control is transferred to the statement following the loop.


                for (int i = 0; i < 10; i++) {
                    if (i == 5) {
                        break; // Exit the loop when i is 5
                    }
                    System.out.println("i: " + i);
                }

Output: This code prints the values of i from 0 to 4. When i reaches 5, the break statement exits the loop.

Continue Statement

The continue statement skips the current iteration of the loop and proceeds to the next iteration. It does not terminate the loop but rather bypasses the remaining code in the current iteration.


                for (int i = 0; i < 10; i++) {
                    if (i % 2 == 0) {
                        continue; // Skip even numbers
                    }
                    System.out.println("Odd number: " + i);
                }

Output: This code prints only the odd numbers from 1 to 9. Even numbers are skipped due to the continue statement.

Break and Continue in Nested Loops

When using break or continue in nested loops, they affect only the innermost loop in which they are called. To break out of multiple nested loops, you might use labeled breaks.


                outerLoop:
                for (int i = 0; i < 5; i++) {
                    for (int j = 0; j < 5; j++) {
                        if (i * j > 6) {
                            break outerLoop; // Exit both loops
                        }
                        System.out.println("i: " + i + ", j: " + j);
                    }
                }

Output: This code prints the values of i and j until the condition i * j > 6 is met, at which point both loops are exited using the labeled break.

Java Object-Oriented Programming (OOP)

Object-Oriented Programming (OOP) is a programming paradigm that uses objects and classes to structure code. Java is an object-oriented language that supports the four main principles of OOP: encapsulation, inheritance, polymorphism, and abstraction.

1. Encapsulation

Encapsulation is the concept of wrapping data (variables) and methods (functions) into a single unit called a class. It helps in hiding the internal state of an object and only exposes a controlled interface to interact with it.


                // Example of encapsulation
                public class Person {
                    // Private variables
                    private String name;
                    private int age;
                    
                    // Public constructor
                    public Person(String name, int age) {
                        this.name = name;
                        this.age = age;
                    }
                    
                    // Public getter for name
                    public String getName() {
                        return name;
                    }
                    
                    // Public setter for name
                    public void setName(String name) {
                        this.name = name;
                    }
                    
                    // Public getter for age
                    public int getAge() {
                        return age;
                    }
                    
                    // Public setter for age
                    public void setAge(int age) {
                        this.age = age;
                    }
                }

2. Inheritance

Inheritance allows a class to inherit properties and methods from another class. This promotes code reusability and establishes a hierarchy between classes.


                // Example of inheritance
                public class Employee extends Person {
                    // Additional property for Employee
                    private double salary;
                    
                    // Constructor for Employee
                    public Employee(String name, int age, double salary) {
                        super(name, age); // Call the constructor of Person class
                        this.salary = salary;
                    }
                    
                    // Getter and setter for salary
                    public double getSalary() {
                        return salary;
                    }
                    
                    public void setSalary(double salary) {
                        this.salary = salary;
                    }
                }

3. Polymorphism

Polymorphism allows methods to have the same name but different implementations based on the object calling the method. It enables one interface to be used for a general class of actions.


                // Example of polymorphism
                public class Animal {
                    public void makeSound() {
                        System.out.println("Some sound...");
                    }
                }
                
                public class Dog extends Animal {
                    @Override
                    public void makeSound() {
                        System.out.println("Bark");
                    }
                }
                
                public class Cat extends Animal {
                    @Override
                    public void makeSound() {
                        System.out.println("Meow");
                    }
                }
                
                // Testing polymorphism
                public class Main {
                    public static void main(String[] args) {
                        Animal myDog = new Dog();
                        Animal myCat = new Cat();
                        
                        myDog.makeSound(); // Outputs: Bark
                        myCat.makeSound(); // Outputs: Meow
                    }
                }

4. Abstraction

Abstraction involves hiding the complex implementation details and showing only the essential features of an object. In Java, abstraction is achieved using abstract classes and interfaces.


                // Example of abstraction using abstract class
                public abstract class Shape {
                    // Abstract method (does not have a body)
                    public abstract void draw();
                    
                    // Regular method
                    public void fillColor() {
                        System.out.println("Filling color...");
                    }
                }
                
                // Concrete subclass
                public class Circle extends Shape {
                    @Override
                    public void draw() {
                        System.out.println("Drawing a circle");
                    }
                }
                
                // Testing abstraction
                public class Main {
                    public static void main(String[] args) {
                        Shape myShape = new Circle();
                        myShape.draw(); // Outputs: Drawing a circle
                        myShape.fillColor(); // Outputs: Filling color...
                    }
                }

Output:

  • Encapsulation demonstrates how to protect and manage data within a class using getters and setters.
  • Inheritance shows how a subclass can extend a superclass, inheriting its properties and methods.
  • Polymorphism enables method overriding, allowing different classes to implement methods with the same name but different behaviors.
  • Abstraction hides the complexities of an implementation and exposes only the necessary parts using abstract classes and methods.

Java Classes and Objects

In Java, classes and objects are fundamental concepts in Object-Oriented Programming (OOP). A class is a blueprint for creating objects, and an object is an instance of a class.

1. Classes

A class is a user-defined data type that contains data members (variables) and member functions (methods) that operate on the data. It defines the properties and behaviors of objects.


                // Example of a class
                public class Car {
                    // Data members (variables)
                    private String brand;
                    private int year;
                    
                    // Constructor
                    public Car(String brand, int year) {
                        this.brand = brand;
                        this.year = year;
                    }
                    
                    // Method to display car details
                    public void displayDetails() {
                        System.out.println("Brand: " + brand);
                        System.out.println("Year: " + year);
                    }
                    
                    // Getter for brand
                    public String getBrand() {
                        return brand;
                    }
                    
                    // Setter for brand
                    public void setBrand(String brand) {
                        this.brand = brand;
                    }
                    
                    // Getter for year
                    public int getYear() {
                        return year;
                    }
                    
                    // Setter for year
                    public void setYear(int year) {
                        this.year = year;
                    }
                }

2. Objects

An object is an instance of a class. It represents a specific example of the class with its own state and behavior. You can create multiple objects from a single class.


                // Example of creating and using objects
                public class Main {
                    public static void main(String[] args) {
                        // Creating an object of the Car class
                        Car myCar = new Car("Toyota", 2020);
                        
                        // Using methods of the Car class
                        myCar.displayDetails();
                        
                        // Modifying the object's data
                        myCar.setYear(2022);
                        System.out.println("Updated Year: " + myCar.getYear());
                    }
                }

Output: - The first part of the code defines a class Car with data members, a constructor, and methods. - The second part demonstrates creating an instance of the Car class, calling its methods, and modifying its data.

Java Class Attributes

In Java, class attributes (also known as data members or fields) are variables that belong to a class. They represent the state or properties of objects created from that class.

1. Declaring Attributes

Attributes are declared within a class and can have different access modifiers to control their visibility. They can be of any data type, including primitive types and objects.


                // Example of declaring class attributes
                public class Person {
                    // Private attributes
                    private String name;
                    private int age;
                    
                    // Public attributes
                    public String address;
                    public double height;
                }

2. Access Modifiers

  • Private: The attribute is accessible only within the class it is declared in. It is used to encapsulate data and enforce data hiding.
  • Public: The attribute is accessible from any other class. It is used for attributes that need to be exposed to other classes.
  • Protected: The attribute is accessible within the same package and by subclasses. (Not shown in the example above)
  • Default (package-private): The attribute is accessible only within its own package. (Not shown in the example above)

3. Example of Using Attributes

Here’s how attributes are used in practice:


                // Example of using class attributes
                public class Main {
                    public static void main(String[] args) {
                        // Creating an object of the Person class
                        Person person = new Person();
                        
                        // Setting attributes using public access
                        person.name = "John Doe";
                        person.age = 30;
                        person.address = "123 Main St";
                        person.height = 5.9;
                        
                        // Displaying attributes
                        System.out.println("Name: " + person.name);
                        System.out.println("Age: " + person.age);
                        System.out.println("Address: " + person.address);
                        System.out.println("Height: " + person.height);
                    }
                }

Output: - The code sets and retrieves values for the attributes of a Person object. - Attributes can be set directly if they are public, and their values can be accessed accordingly.

Java Constructors

In Java, a constructor is a special method that is used to initialize objects. It is called when an object of a class is created and has the same name as the class. Constructors do not have a return type, not even void.

Types of Constructors

  • Default Constructor: A constructor with no parameters. If no constructor is explicitly defined in the class, the Java compiler provides a default constructor that initializes all member variables to their default values.
  • Parameterized Constructor: A constructor that takes parameters. It allows initializing objects with specific values at the time of creation.

Syntax


                // Default Constructor
                public class Person {
                    String name;
                    int age;
                    
                    // Default constructor
                    public Person() {
                        name = "Unknown";
                        age = 0;
                    }
                }
                
                // Parameterized Constructor
                public class Person {
                    String name;
                    int age;
                    
                    // Parameterized constructor
                    public Person(String name, int age) {
                        this.name = name;
                        this.age = age;
                    }
                }

Examples

Example: Default Constructor


                // Using the default constructor
                public class Main {
                    public static void main(String[] args) {
                        // Creating an object using the default constructor
                        Person person = new Person();
                        System.out.println("Name: " + person.name);
                        System.out.println("Age: " + person.age);
                    }
                }

Example: Parameterized Constructor


                // Using the parameterized constructor
                public class Main {
                    public static void main(String[] args) {
                        // Creating an object using the parameterized constructor
                        Person person = new Person("John Doe", 30);
                        System.out.println("Name: " + person.name);
                        System.out.println("Age: " + person.age);
                    }
                }

Output: - The default constructor initializes the name to "Unknown" and age to 0. - The parameterized constructor initializes the name and age with the provided values.
