Python Introductions

History of Python

Python was created by Guido van Rossum and first released in 1991. It was developed as a successor to the ABC programming language, which was capable of exception handling and interfacing with the Amoeba operating system.

Introduction to Python

Python is a high-level, interpreted programming language known for its ease of learning and readability. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming. Python is often used for web development, automation, data analysis, artificial intelligence, and more.

How Python Works

Python code is interpreted by the Python interpreter, which executes the code line by line. Python is cross-platform, meaning that the same Python code can run on different operating systems with little or no modification. Python's simple syntax and dynamic typing make it an ideal language for beginners and experts alike.

Python Syntax

Indentation

Indentation in Python is used to define the structure of the code. Unlike other programming languages that use curly braces or other markers, Python relies on indentation to determine the grouping of statements.

                if True:
            print("This is indented")
            print("This is also part of the if block")

In the above example, the code inside the if statement is indented to define a block of code that should be executed when the condition is true.

Python Comments

Comments in Python are used to explain code and are not executed by the interpreter. They are marked with a # symbol for single-line comments, or triple quotes """ or ''' for multi-line comments.

                # This is a single-line comment
        """
        This is a multi-line comment
        spanning multiple lines
        """

The # symbol starts a single-line comment, and anything following it on the line is ignored by the interpreter. Multi-line comments are enclosed in triple quotes.

Python Variables

Variable Assignment

Variables in Python are used to store data values. You do not need to declare variables before using them, and you can assign a value to a variable using the = operator.

Python Casting

Casting in Python

Casting is the process of converting a variable from one type to another. Python has built-in functions for casting types, including int(), float(), and str().

    x = int(3.14)
    print(x)  # Outputs: 3

    # Convert an integer to a float
    y = float(10)
    print(y)  # Outputs: 10.0
    print(type(y))  # Outputs: <class 'float'>
    
    # Convert a number to a string
    z = str(100)
    print(z)  # Outputs: '100'
    print(type(z))  # Outputs: <class 'str'>

Casting to Complex Numbers

You can cast numbers to complex using the complex() function.

    # Convert a float to a complex number

If ... Else Statement

      
# Example of if ... else statement

  print("x is greater than 5")
else:
  print("x is not greater than 5")

Output: x is not greater than 5

If ... Elif ... Else Statement

      
# Example of if ... elif ... else statement

if x > 10:
  print("x is greater than 10")

  print("x is equal to 7")

Python Tuples

A tuple in Python is an immutable, ordered collection of elements. Tuples are similar to lists, but unlike lists, the elements of a tuple cannot be changed once they are assigned.

Basic Python Tuple Example

        
  # Example of a basic Python tuple
  my_tuple = (1, 2, 3, "Hello", 5.5)
  print(my_tuple)

Output: (1, 2, 3, 'Hello', 5.5)

What is a Python Set?

A Python set is an unordered collection of unique elements. Sets are used when you want to store multiple items in a single variable, but unlike lists or tuples, they do not allow duplicate values.

Sets are mutable, meaning you can add or remove items from them, but the elements themselves must be immutable (e.g., strings, numbers, or tuples).

  • Unique Elements: Sets automatically remove duplicate entries.
  • Unordered Collection: The elements in a set have no specific order.
  • Efficient Membership Testing: Sets are optimized for fast membership testing, making them ideal for use cases where you need to check if an item is in the set.
  • Set Operations: Python provides built-in operations for sets, such as union, intersection, difference, and symmetric difference.

Python Set Example

# Example of a basic Python set
  my_set = {1, 2, 3, "Hello", 5.5, 2}  # Duplicate '2' will be removed
  print(my_set)
  
  # Adding an element to the set
  my_set.add("World")
  print(my_set)
  
  # Output: {1, 2, 3, 'Hello', 5.5}
  # Output after adding 'World': {1, 2, 3, 'Hello', 5.5, 'World'}

Python Dictionaries

A Python dictionary is an unordered, mutable, and indexed collection of key-value pairs. Each key in a dictionary must be unique, and it is used to access its associated value.

  • Keys: Keys are unique identifiers for elements in the dictionary. They can be of any immutable type, such as strings, numbers, or tuples.
  • Values: Values can be of any data type and can be repeated. They are the data associated with each key.
  • Mutability: Dictionaries are mutable, meaning that their contents can be changed after creation. You can add, remove, or modify key-value pairs.
  • Unordered: Dictionaries do not maintain order, although in Python 3.7 and later, dictionaries maintain the insertion order.

The general syntax for a Python dictionary is:

my_dict = {'key1': 'value1', 'key2': 'value2'}

Example of a Python Dictionary

# Example of a basic Python dictionary
  my_dict = {
      "name": "ElightenDrashta",
      "type": "Tutorial",
      "topics": ["Python", "JavaScript", "CSS"]
  }
  
  # Accessing a value by its key
  print(my_dict["name"])  # Output: ElightenDrashta
  
  # Adding a new key-value pair
  my_dict["difficulty"] = "Beginner"
  print(my_dict)
  
  # Output:
  # {'name': 'ElightenDrashta', 'type': 'Tutorial', 'topics': ['Python', 'JavaScript', 'CSS'], 'difficulty': 'Beginner'}

Python While Loops

A while loop in Python repeatedly executes a block of code as long as a given condition is true. It's a fundamental looping construct in Python and is useful when the number of iterations is not known in advance.

  • Condition: The loop continues to execute as long as the condition evaluates to True. Once the condition becomes False, the loop stops.
  • Loop Body: The code block inside the loop that runs on each iteration as long as the condition is True.
  • Infinite Loop: Be cautious of loops where the condition never becomes False, as this will create an infinite loop.

The general syntax for a while loop is:

while condition:
      # Code block to be executed

Example of a Python While Loop

# Example of a basic Python while loop
  counter = 0
  
  while counter < 5:
      print("Counter is at:", counter)
      counter += 1
  
  # Output:
  # Counter is at: 0
  # Counter is at: 1
  # Counter is at: 2
  # Counter is at: 3
  # Counter is at: 4

This example demonstrates a simple while loop that prints the value of the counter variable and increments it on each iteration. The loop runs until the counter reaches 5.

Python For Loops

Python's for loop is used to iterate over a sequence (like a list, tuple, dictionary, set, or string) or other iterable objects. With the for loop, we can execute a set of statements, once for each item in a sequence.

Basic Syntax

Here’s the basic syntax of a for loop in Python:

for item in sequence:
      # Execute statements

Example: Iterating Through a List

Below is an example of a for loop that iterates through a list of numbers and prints each one:

numbers = [1, 2, 3, 4, 5]
  for num in numbers:
      print(num)

Example: Using range() in a For Loop

You can use the range() function to generate a sequence of numbers, which can be useful in loops:

for i in range(5):
      print("Iteration", i)

Python Functions

Functions in Python are blocks of reusable code that perform a specific task. Functions can accept inputs, called parameters, and return an output. They help in organizing code and making it more modular and maintainable.

Defining a Function

You can define a function in Python using the def keyword, followed by the function name and parentheses:

def function_name(parameters):
          # Function body
          return result

Example: A Simple Function

Here’s an example of a simple function that adds two numbers:

def add_numbers(a, b):
          return a + b
      
      result = add_numbers(3, 5)
      print(result)  # Output will be 8

Calling a Function

Once a function is defined, you can call it by using its name followed by parentheses. If the function takes parameters, you pass the arguments inside the parentheses:

result = function_name(arguments)

Python Lambda Functions

A lambda function in Python is a small, anonymous function defined with the lambda keyword. These functions can have any number of arguments but only one expression. They are often used for short, simple functions that are not reused elsewhere.

Syntax of a Lambda Function

The basic syntax of a lambda function is:

lambda arguments: expression

Example: Simple Lambda Function

Here’s an example of a simple lambda function that adds 10 to a given number:

add_10 = lambda x: x + 10
      print(add_10(5))  # Output will be 15

Example: Lambda with Multiple Arguments

Lambdas can also take multiple arguments. Below is an example of a lambda function that adds two numbers:

add_numbers = lambda a, b: a + b
      print(add_numbers(3, 5))  # Output will be 8

Use Case: Sorting with Lambda

One common use of lambda functions is in functions like sort() or sorted() when you need to specify a custom sorting order:

points = [(2, 3), (1, 5), (4, 1)]
      points_sorted = sorted(points, key=lambda point: point[1])
      print(points_sorted)  # Output will be [(4, 1), (2, 3), (1, 5)]

Python Arrays

In Python, arrays are a collection of items stored at contiguous memory locations. They are used to store multiple values in a single variable. Arrays in Python can be implemented using the array module or using lists (though lists are more commonly used in Python).

Here is an example of how to create and manipulate arrays in Python:

# Example of a Python array using the array module
      import array
      
      # Creating an array of integers
      arr = array.array('i', [1, 2, 3, 4, 5])
      
      # Accessing elements of the array
      print("First element:", arr[0])
      
      # Appending a new element
      arr.append(6)
      print("Array after appending:", arr)
      
      # Removing an element
      arr.remove(3)
      print("Array after removing an element:", arr)

Output:

First element: 1
      Array after appending: array('i', [1, 2, 3, 4, 5, 6])
      Array after removing an element: array('i', [1, 2, 4, 5, 6])
